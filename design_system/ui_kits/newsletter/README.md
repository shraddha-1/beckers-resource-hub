# Newsletter — UI Kit

A recreation of the Becker's daily email newsletter. 640px wide, editorial stack, red rule under the masthead, navy footer.

## Files
- `index.html` — rendered sample newsletter.
- `Newsletter.jsx` — `NewsletterMasthead`, `NewsletterHero`, `NewsletterItem`, `NewsletterSection`, `NewsletterFooter`.
- `kit.css` — scoped styles. Depends on `../../colors_and_type.css`.

## Fidelity notes
- Item copy is pulled from recent Becker's headlines to match voice.
- Real emails should use tables (not divs) for Outlook compatibility; this kit mocks the visual structure only.
